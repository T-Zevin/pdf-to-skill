## Migration Protocol

1. Define the target question in the same logical form as the original paper.
2. Map original variables, samples, and endpoints to target-direction equivalents.
3. Rebuild the minimum evidence chain: discovery -> validation -> mechanism/utility -> robustness.
4. Preserve the original method's decision points but reset thresholds to the new dataset/domain.
5. Write limitations before claims to avoid over-transfer.
